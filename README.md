# EEG-based ADHD Classification

## Overview
This project focuses on classifying ADHD vs Control subjects using EEG signals and machine learning techniques.

## Dataset
- Multi-subject EEG dataset
- 2 channels
- Labels: 0 (Control), 1 (ADHD)

## Methodology
- Signal preprocessing
- Feature extraction
- Machine learning models

(87.66%)
